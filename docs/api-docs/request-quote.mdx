## Quote Best

Using the quote method, you can get a preview of the best route for this cross-chain swap. It goes through all the possible DEXs, bridges and aggregators to find the best possible single-step route based on user experience, fee amount, and swap output. You can check available request params [here](#quote-request-params).

Here is an example of quoting swap 1.05 ETH in Ethereum Blockchain for Matic in Polygon Blockchain

<Tabs>
<Tab title="Node.JS (Axios) Request">

```typescript
const response = await axios.post('https://api-v2.rubic.exchange/api/routes/quoteBest', {
</strong>  params: {
    "dstTokenAddress": "0x0000000000000000000000000000000000000000",
    "dstTokenBlockchain": "POLYGON",
    "referrer": "rubic.exchange",
    "srcTokenAddress": "0x0000000000000000000000000000000000000000",
    "srcTokenAmount": "1.05",
    "srcTokenBlockchain": "ETH"
  }
});
```

</Tab>

## Quote All

Using the quote method, you can get a preview of all possible routes for this cross-chain swap. It goes through all the possible DEXs, bridges and aggregators to find and provide all single-step routes based on user experience, fee amount, and swap output. You can check available request params [here](#quote-request-params).

Here is an example of quoting swap 1.05 ETH in Ethereum Blockchain for Matic in Polygon Blockchain

<Tabs>
<Tab title="Node.JS (Axios) Request">

```typescript
const response = await axios.post('https://api-v2.rubic.exchange/api/routes/quoteAll', {
</strong>  params: {
    "dstTokenAddress": "0x0000000000000000000000000000000000000000",
    "dstTokenBlockchain": "POLYGON",
    "referrer": "rubic.exchange",
    "srcTokenAddress": "0x0000000000000000000000000000000000000000",
    "srcTokenAmount": "1.05",
    "srcTokenBlockchain": "ETH"
  }
});
```

</Tab>
</Tabs>

[Open Rubic Swagger](https://api-v2.rubic.exchange/api/swagger/#/Router/RouterController_getQuoteAll)
