## Cross-Chain Token Swaps

> Swap 500 USDC from Ethereum to Arbitrum

Ask an AI agent to swap tokens between supported chains, compare routes, simulate the outcome, and proceed only after approval.

## On-Chain Token Swaps

> Swap ETH to USDC on Arbitrum

Use Rubic’s MCP Server for same-chain token swaps where supported, with quoting, simulation, and transaction preparation via MCP tools.

## Route Comparison with

> Show me all routes for 1 ETH to MATIC

Compare swap routes by estimated output, fees, providers, duration, and execution risk before choosing how to proceed.

## Swap Simulation with

> Simulate swapping 1 AVAX to ETH

Preview expected output, total estimated cost, gas, price impact, route details, and risk before anything is signed or broadcasted.

## Portfolio and Balance Checks

> Check my balances across supported chains

Check token balances across supported EVM chains so the agent can understand available assets before building a swap.

## Browser Fallback with

> Open this swap route in Rubic

Generate a Rubic URL when users need manual confirmation, non-EVM signing, or a browser-based fallback flow.
