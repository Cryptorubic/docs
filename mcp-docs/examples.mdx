## **Get Token Details**

> Get USDC token info on Ethereum

**Response:**

```json
{
	"address": "0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48",
	"decimals": 6,
	"symbol": "USDC",
	"name": "USD Coin",
	"blockchain": "ETH",
	"price": 0.999646
}
```
